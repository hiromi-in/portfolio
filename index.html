Hello!

I am Hiromi Ishikawa, a Bilingual Engineer at Qualitest supporting Google in Austin, Texas. I used to be a software sales at SAP in Tokyo, Japan. In 2021, I decided to change my career path and go into the technical field which I was always interested in since when I was at SAP.

Being in the technical area with some sales background gave me interesting perspectives as I can see things from a front end perspective(selling) and a back end perspective(technical). As a sales person, I always think about how to show information and convey messages. When I was selling, I completely relied on software to communicate publicly with people. Since I joined Qualitest, I realized that I can make something to do it by myself.

So I started learning coding. Since my current job is data engineering, I started from SQL, Python and R for data analysis and visualization. Then I encountered HTML and CSS while I was working on data scraping with Python. I fell in love with them. I loved the limitless potential of web development. JavaScript was a great addition to my web development journey.
